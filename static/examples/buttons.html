Buttons

Button .c-btn .c-btn--primary

Button .c-btn .c-btn--secondary

The button component needs an extension (--primary, --secondary, --danger) in order have a background color applied

The primary button is used for the user to navigate through the pages, load more assets etc.

The secondary button is used for actions that affect the database.

  • Create new patient
  • Edit information
  • Submit a form
  • etc
Button .c-btn .c-btn--primary .c-btn--large

Button .c-btn .c-btn--primary .c-btn--small

The size variations are also extensions to the main button component (--small, --large)

Button .c-btn .c-btn--primary .c-btn--large .c-btn--full

The --full extension allows the button to expand to the width of its parent.

Button .c-btn .c-btn--primary .c-btn--large .c-btn--ghost Button .c-btn .c-btn--secondary .c-btn--small .c-btn--ghost

The ghost button removes the background and adds a border. It can be combined with all the variations we have already seen

Button .c-btn .c-btn--danger .c-btn--large

The danger button cannot be in ghost state. With this extension we can only control the size
